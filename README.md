# Claude Code Skills Marketplace

Personal collection of Claude Code plugins.

## Installation

Add this marketplace:

```bash
claude plugin marketplace add hansharhoff/skills
```

Then install any plugin:

```bash
claude plugin install teach-tech@hansharhoff/skills
```

Or browse available plugins:

```bash
claude plugin discover
```

## Plugins

- **teach-tech** — Interactive technology teaching sessions with guided examples and hands-on exercises
- **repo-cleanup** — Audit and clean up a repository for quality issues with prioritized reporting
