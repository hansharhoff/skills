# teach-tech

A Claude Code skill that helps you learn new technologies through guided, interactive teaching sessions.

## What it does

When you ask Claude Code to teach you a new technology, this skill:

- Asks clarifying questions about your goals and which aspects to focus on
- Researches best practices and relates concepts to tools you already know
- Creates a structured learning plan for your approval
- Uses diagrams, examples, and hands-on exercises
- Encourages active learning — pauses for you to reflect and practice

## Usage

```
/teach-tech <technology>
```

Or just ask Claude Code to teach you something — the skill activates based on context.
