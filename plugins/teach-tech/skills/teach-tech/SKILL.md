---
name: teach-tech
description: Teach the user a new technology by guiding them through simple examples and letting them do some of the work. Use it when the user want to learn a new thing.
---

When the user ask the learn a new technology we should help them by teaching them the concepts of the tecnology using explanations, diagrams and examples they can work through themselves.

Start by asking the user some clarifying questions why they want to learn the tecnology and provide some multiple choice options for why and which aspects.

Then perform a web research for best practices within this tool and use other skills to understand the usual tech that the user usually uses and how it relates to the technology. Use the existing knowledge as examples to help understand the new tech.

Make sure that you make a plan for how the user should learn the technology and explain it to the user. Then ask for confirimation before starting.

Make sure to use diagrams where appropriate and ask the user to execute various terminal commands and paste relevant code snippets into files so that they do not zone out completely. Assume that we can use a sandbox folder in the current folder for small examples.

Suggest to the users that they actively learn not passively. Stop generating whenever it is time for the user to reflect on the material or do something. Then continue when the users reqeusts that you do so.
